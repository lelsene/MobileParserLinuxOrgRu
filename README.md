# MobileParserLinuxOrgRu
Мобильное приложение, агрегатор новостей сайта linux.org.ru

## Функциональные возможности
1. Просмотр списка последних новостей, отсортированный по дате публикации, с краткой информацией: автор, название, часть статьи, дата публикации, теги.
2. Просмотр полной версии новости с комментариями.
3. Добавление новости в закладки для офлайн просмотра.
4. Добавление (по нажатию на тег новости, по нажатию на полосу тегов - своего собственного) тега для отображения новостей с таким тегом в персональной подборке. 
5. Просмотр персональной подборки по тегам.

- Room 2.2.5
- Gson 2.8.6
- CardView 1.0.0

- SwipeRefreshLayout 1.1.0
